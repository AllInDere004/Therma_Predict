# ThermaPredict

## 🚀 AI-Based Predictive Maintenance System

ThermaPredict is a machine learning-based predictive maintenance system that monitors industrial machine health and predicts possible failures before they occur.

## Project Status
🚧 Currently Under Development

## Features
- Machine Failure Prediction
- Health Score Calculation
- Interactive Dashboard
- Device Monitoring
- Flask REST API
- React Frontend
- Machine Learning using AdaBoost

## Tech Stack
- React.js
- Tailwind CSS
- Flask
- Python
- Pandas
- Scikit-learn
- AdaBoost
- Git & GitHub
